## What I learned
- Agent skill may be right way 
  - what exactly is a skill: 
    - the system prompt written in file
    - which can be load from outside of the agent (container)
    - structured, easy for both human and agent to read
  - replace MCP well in some cases
  - flexible, easy to read and edit
- Context is the key for agent
  - stay focused
  - use multiple agents when necessary
  - Give your agents all they need, but let themselve to digg into them
- Vibe coding is dead - let the agent write the code!

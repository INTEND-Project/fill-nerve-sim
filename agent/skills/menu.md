# Skills menu

Add skill folders under `skills/` (each with a `SKILL.md`) to describe available APIs or workflows.
The agent reloads this menu and all skill files on every user prompt.

## FILL machine data analytics control

FILL machine data analytics provide REST API to check machines, workloads and there versions, and deploy/undeploy workloads on the machines.

SKILL location: fill-machines/SKILL.md
